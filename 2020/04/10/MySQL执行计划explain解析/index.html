    • idзӣёеҗҢж—¶пјҢжү§иЎҢйЎәеәҸз”ұдёҠиҮідёӢ
    • еҰӮжһңжҳҜеӯҗжҹҘиҜўпјҢidзҡ„еәҸеҸ·дјҡйҖ’еўһпјҢidеҖји¶ҠеӨ§дјҳе…Ҳзә§и¶Ҡй«ҳпјҢи¶Ҡе…Ҳиў«жү§иЎҢ
    • idеҰӮжһңзӣёеҗҢпјҢеҸҜд»Ҙи®ӨдёәжҳҜдёҖз»„пјҢд»ҺдёҠеҫҖдёӢйЎәеәҸжү§иЎҢпјӣеңЁжүҖжңүз»„дёӯпјҢidеҖји¶ҠеӨ§пјҢдјҳе…Ҳзә§и¶Ҡй«ҳпјҢи¶Ҡе…Ҳжү§иЎҢ

  • select_typeпјҡжҹҘиҜўдёӯжҜҸдёӘselectеӯҗеҸҘзҡ„зұ»еһӢ

    • SIMPLE(з®ҖеҚ•SELECT,дёҚдҪҝз”ЁUNIONжҲ–еӯҗжҹҘиҜўзӯү)

    • PRIMARY(жҹҘиҜўдёӯиӢҘеҢ…еҗ«д»»дҪ•еӨҚжқӮзҡ„еӯҗйғЁеҲҶ,жңҖеӨ–еұӮзҡ„selectиў«ж Үи®°дёәPRIMARY)

    • UNION(UNIONдёӯзҡ„з¬¬дәҢдёӘжҲ–еҗҺйқўзҡ„SELECTиҜӯеҸҘ)

    • DEPENDENT UNION(UNIONдёӯзҡ„з¬¬дәҢдёӘжҲ–еҗҺйқўзҡ„SELECTиҜӯеҸҘпјҢеҸ–еҶідәҺеӨ–йқўзҡ„жҹҘиҜў)

    • UNION RESULT(UNIONзҡ„з»“жһң)

    • SUBQUERY(еӯҗжҹҘиҜўдёӯзҡ„з¬¬дёҖдёӘSELECT)

    • DEPENDENT SUBQUERY(еӯҗжҹҘиҜўдёӯзҡ„з¬¬дёҖдёӘSELECTпјҢеҸ–еҶідәҺеӨ–йқўзҡ„жҹҘиҜў)

    • DERIVED(жҙҫз”ҹиЎЁзҡ„SELECT, FROMеӯҗеҸҘзҡ„еӯҗжҹҘиҜў)

    • UNCACHEABLE SUBQUERY(дёҖдёӘеӯҗжҹҘиҜўзҡ„з»“жһңдёҚиғҪиў«зј“еӯҳпјҢеҝ…йЎ»йҮҚж–°иҜ„дј°еӨ–й“ҫжҺҘзҡ„з¬¬дёҖиЎҢ)

  • tableпјҡжҳҫзӨәиҝҷдёҖиЎҢзҡ„ж•°жҚ®жҳҜе…ідәҺе“Әеј иЎЁзҡ„пјҢжңүж—¶дёҚжҳҜзңҹе®һзҡ„иЎЁеҗҚеӯ—,зңӢеҲ°зҡ„жҳҜderivedx(xжҳҜдёӘж•°еӯ—,жҲ‘зҡ„зҗҶи§ЈжҳҜз¬¬еҮ жӯҘжү§иЎҢзҡ„з»“жһң)

  • partitionsпјҡеҲҶеҢәеҸ·

  • typeпјҡиЎЁзӨәMySQLеңЁиЎЁдёӯжүҫеҲ°жүҖйңҖиЎҢзҡ„ж–№ејҸпјҢеҸҲз§°вҖңи®ҝй—®зұ»еһӢвҖқпјҢеёёз”Ёзҡ„зұ»еһӢжңүпјҡ ALL, index, range, ref, eq_ref, const, system, NULLпјҲд»Һе·ҰеҲ°еҸіпјҢжҖ§иғҪд»Һе·®еҲ°еҘҪпјү

    • ALLпјҡFull Table ScanпјҢ MySQLе°ҶйҒҚеҺҶе…ЁиЎЁд»ҘжүҫеҲ°еҢ№й…Қзҡ„иЎҢ
    • indexпјҡFull Index ScanпјҢindexдёҺALLеҢәеҲ«дёәindexзұ»еһӢеҸӘйҒҚеҺҶзҙўеј•ж ‘
    • rangeпјҡеҸӘжЈҖзҙўз»ҷе®ҡиҢғеӣҙзҡ„иЎҢпјҢдҪҝз”ЁдёҖдёӘзҙўеј•жқҘйҖүжӢ©иЎҢ
    • refпјҡиЎЁзӨәдёҠиҝ°иЎЁзҡ„иҝһжҺҘеҢ№й…ҚжқЎд»¶пјҢеҚіе“ӘдәӣеҲ—жҲ–еёёйҮҸиў«з”ЁдәҺжҹҘжүҫзҙўеј•еҲ—дёҠзҡ„еҖј
    • eq_refпјҡзұ»дјјrefпјҢеҢәеҲ«е°ұеңЁдҪҝз”Ёзҡ„зҙўеј•жҳҜе”ҜдёҖзҙўеј•пјҢеҜ№дәҺжҜҸдёӘзҙўеј•й”®еҖјпјҢиЎЁдёӯеҸӘжңүдёҖжқЎи®°еҪ•еҢ№й…ҚпјҢз®ҖеҚ•жқҘиҜҙпјҢе°ұжҳҜеӨҡиЎЁиҝһжҺҘдёӯдҪҝз”Ёprimary keyжҲ–иҖ… unique keyдҪңдёәе…іиҒ”жқЎд»¶
    • constгҖҒsystemпјҡеҪ“MySQLеҜ№жҹҘиҜўжҹҗйғЁеҲҶиҝӣиЎҢдјҳеҢ–пјҢе№¶иҪ¬жҚўдёәдёҖдёӘеёёйҮҸж—¶пјҢдҪҝз”Ёиҝҷдәӣзұ»еһӢи®ҝй—®гҖӮеҰӮе°Ҷдё»й”®зҪ®дәҺwhereеҲ—иЎЁдёӯпјҢMySQLе°ұиғҪе°ҶиҜҘжҹҘиҜўиҪ¬жҚўдёәдёҖдёӘеёёйҮҸ,systemжҳҜconstзұ»еһӢзҡ„зү№дҫӢпјҢеҪ“жҹҘиҜўзҡ„иЎЁеҸӘжңүдёҖиЎҢзҡ„жғ…еҶөдёӢпјҢдҪҝз”Ёsystem
    • NULLпјҡMySQLеңЁдјҳеҢ–иҝҮзЁӢдёӯеҲҶи§ЈиҜӯеҸҘпјҢжү§иЎҢж—¶з”ҡиҮідёҚз”Ёи®ҝй—®иЎЁжҲ–зҙўеј•пјҢдҫӢеҰӮд»ҺдёҖдёӘзҙўеј•еҲ—йҮҢйҖүеҸ–жңҖе°ҸеҖјеҸҜд»ҘйҖҡиҝҮеҚ•зӢ¬зҙўеј•жҹҘжүҫе®ҢжҲҗгҖӮ

  • possible_keysпјҡжҢҮеҮәMySQLиғҪдҪҝз”Ёе“ӘдёӘзҙўеј•еңЁиЎЁдёӯжүҫеҲ°и®°еҪ•пјҢжҹҘиҜўж¶үеҸҠеҲ°зҡ„еӯ—ж®өдёҠиӢҘеӯҳеңЁзҙўеј•пјҢеҲҷиҜҘзҙўеј•е°Ҷиў«еҲ—еҮәпјҢдҪҶдёҚдёҖе®ҡиў«жҹҘиҜўдҪҝз”Ё

    иҜҘеҲ—е®Ңе…ЁзӢ¬з«ӢдәҺEXPLAINиҫ“еҮәжүҖзӨәзҡ„иЎЁзҡ„ж¬ЎеәҸгҖӮиҝҷж„Ҹе‘ізқҖеңЁpossible_keysдёӯзҡ„жҹҗдәӣй”®е®һйҷ…дёҠдёҚиғҪжҢүз”ҹжҲҗзҡ„иЎЁж¬ЎеәҸдҪҝз”ЁгҖӮеҰӮжһңиҜҘеҲ—жҳҜNULLпјҢеҲҷжІЎжңүзӣёе…ізҡ„зҙўеј•гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢеҸҜд»ҘйҖҡиҝҮжЈҖжҹҘWHEREеӯҗеҸҘзңӢжҳҜеҗҰе®ғеј•з”ЁжҹҗдәӣеҲ—жҲ–йҖӮеҗҲзҙўеј•зҡ„еҲ—жқҘжҸҗй«ҳдҪ зҡ„жҹҘиҜўжҖ§иғҪгҖӮеҰӮжһңжҳҜиҝҷж ·пјҢеҲӣйҖ дёҖдёӘйҖӮеҪ“зҡ„зҙўеј•е№¶дё”еҶҚж¬Ўз”ЁEXPLAINжЈҖжҹҘжҹҘиҜў

  • KeyпјҡжҳҫзӨәMySQLе®һйҷ…еҶіе®ҡдҪҝз”Ёзҡ„й”®пјҲзҙўеј•пјү

    еҰӮжһңжІЎжңүйҖүжӢ©зҙўеј•пјҢй”®жҳҜNULLгҖӮиҰҒжғіејәеҲ¶MySQLдҪҝз”ЁжҲ–еҝҪи§Ҷpossible_keysеҲ—дёӯзҡ„зҙўеј•пјҢеңЁжҹҘиҜўдёӯдҪҝз”ЁFORCE INDEXгҖҒUSE INDEXжҲ–иҖ…IGNORE INDEXгҖӮ

  • key_lenпјҡиЎЁзӨәзҙўеј•дёӯдҪҝз”Ёзҡ„еӯ—иҠӮж•°пјҢеҸҜйҖҡиҝҮиҜҘеҲ—и®Ўз®—жҹҘиҜўдёӯдҪҝз”Ёзҡ„зҙўеј•зҡ„й•ҝеәҰпјҲkey_lenжҳҫзӨәзҡ„еҖјдёәзҙўеј•еӯ—ж®өзҡ„жңҖеӨ§еҸҜиғҪй•ҝеәҰпјҢе№¶йқһе®һйҷ…дҪҝз”Ёй•ҝеәҰпјҢеҚіkey_lenжҳҜж №жҚ®иЎЁе®ҡд№үи®Ўз®—иҖҢеҫ—пјҢдёҚжҳҜйҖҡиҝҮиЎЁеҶ…жЈҖзҙўеҮәзҡ„пјү

    дёҚжҚҹеӨұзІҫзЎ®жҖ§зҡ„жғ…еҶөдёӢпјҢй•ҝеәҰи¶Ҡзҹӯи¶ҠеҘҪ

  • refпјҡиЎЁзӨәдёҠиҝ°иЎЁзҡ„иҝһжҺҘеҢ№й…ҚжқЎд»¶пјҢеҚіе“ӘдәӣеҲ—жҲ–еёёйҮҸиў«з”ЁдәҺжҹҘжүҫзҙўеј•еҲ—дёҠзҡ„еҖј

  • rowsпјҡиЎЁзӨәMySQLж №жҚ®иЎЁз»ҹи®ЎдҝЎжҒҜеҸҠзҙўеј•йҖүз”Ёжғ…еҶөпјҢдј°з®—зҡ„жүҫеҲ°жүҖйңҖзҡ„и®°еҪ•жүҖйңҖиҰҒиҜ»еҸ–зҡ„иЎҢж•°

  • ExtraпјҡиҜҘеҲ—еҢ…еҗ«MySQLи§ЈеҶіжҹҘиҜўзҡ„иҜҰз»ҶдҝЎжҒҜ,жңүд»ҘдёӢеҮ з§Қжғ…еҶөпјҡ

    • Using filesort

      MySQLжңүдёӨз§Қж–№ејҸеҸҜд»Ҙз”ҹжҲҗжңүеәҸзҡ„з»“жһңпјҢйҖҡиҝҮжҺ’еәҸж“ҚдҪңжҲ–иҖ…дҪҝз”Ёзҙўеј•пјҢеҪ“ExtraдёӯеҮәзҺ°дәҶUsing filesort иҜҙжҳҺMySQLдҪҝз”ЁдәҶеҗҺиҖ…пјҢдҪҶжіЁж„ҸиҷҪз„¶еҸ«filesortдҪҶе№¶дёҚжҳҜиҜҙжҳҺе°ұжҳҜз”ЁдәҶж–Үд»¶жқҘиҝӣиЎҢжҺ’еәҸпјҢеҸӘиҰҒеҸҜиғҪжҺ’еәҸйғҪжҳҜеңЁеҶ…еӯҳйҮҢе®ҢжҲҗзҡ„гҖӮеӨ§йғЁеҲҶжғ…еҶөдёӢеҲ©з”Ёзҙўеј•жҺ’еәҸжӣҙеҝ«пјҢжүҖд»ҘдёҖиҲ¬иҝҷж—¶д№ҹиҰҒиҖғиҷ‘дјҳеҢ–жҹҘиҜўдәҶгҖӮдҪҝз”Ёж–Үд»¶е®ҢжҲҗжҺ’еәҸж“ҚдҪңпјҢиҝҷжҳҜеҸҜиғҪжҳҜordery byпјҢgroup byиҜӯеҸҘзҡ„з»“жһңпјҢиҝҷеҸҜиғҪжҳҜдёҖдёӘCPUеҜҶйӣҶеһӢзҡ„иҝҮзЁӢпјҢеҸҜд»ҘйҖҡиҝҮйҖүжӢ©еҗҲйҖӮзҡ„зҙўеј•жқҘж”№иҝӣжҖ§иғҪпјҢз”Ёзҙўеј•жқҘдёәжҹҘиҜўз»“жһңжҺ’еәҸгҖӮ

    • Using temporary

      з”Ёдёҙж—¶иЎЁдҝқеӯҳдёӯй—ҙз»“жһңпјҢеёёз”ЁдәҺGROUP BY е’Ң ORDER BYж“ҚдҪңдёӯпјҢдёҖиҲ¬зңӢеҲ°е®ғиҜҙжҳҺжҹҘиҜўйңҖиҰҒдјҳеҢ–дәҶпјҢе°ұз®—йҒҝе…ҚдёҚдәҶдёҙж—¶иЎЁзҡ„дҪҝз”Ёд№ҹиҰҒе°ҪйҮҸйҒҝе…ҚзЎ¬зӣҳдёҙж—¶иЎЁзҡ„дҪҝз”ЁгҖӮ

    • Not exists

      MYSQLдјҳеҢ–дәҶLEFT JOINпјҢдёҖж—Ұе®ғжүҫеҲ°дәҶеҢ№й…ҚLEFT JOINж ҮеҮҶзҡ„иЎҢпјҢ е°ұдёҚеҶҚжҗңзҙўдәҶ

    • Using index

      иҜҙжҳҺжҹҘиҜўжҳҜиҰҶзӣ–дәҶзҙўеј•зҡ„пјҢдёҚйңҖиҰҒиҜ»еҸ–ж•°жҚ®ж–Үд»¶пјҢд»Һзҙўеј•ж ‘пјҲзҙўеј•ж–Үд»¶пјүдёӯеҚіеҸҜиҺ·еҫ—дҝЎжҒҜгҖӮеҰӮжһңеҗҢж—¶еҮәзҺ°using whereпјҢиЎЁжҳҺзҙўеј•иў«з”ЁжқҘжү§иЎҢзҙўеј•й”®еҖјзҡ„жҹҘжүҫпјҢжІЎжңүusing whereпјҢиЎЁжҳҺзҙўеј•з”ЁжқҘиҜ»еҸ–ж•°жҚ®иҖҢйқһжү§иЎҢжҹҘжүҫеҠЁдҪңгҖӮиҝҷжҳҜMySQLжңҚеҠЎеұӮе®ҢжҲҗзҡ„пјҢдҪҶж— йңҖеҶҚеӣһиЎЁжҹҘиҜўи®°еҪ•гҖӮ

    • Using index condition

      иҝҷжҳҜMySQL 5.6еҮәжқҘзҡ„ж–°зү№жҖ§пјҢеҸ«еҒҡвҖңзҙўеј•жқЎд»¶жҺЁйҖҒвҖқгҖӮз®ҖеҚ•иҜҙдёҖзӮ№е°ұжҳҜMySQLеҺҹжқҘеңЁзҙўеј•дёҠжҳҜдёҚиғҪжү§иЎҢеҰӮlikeиҝҷж ·зҡ„ж“ҚдҪңзҡ„пјҢдҪҶжҳҜзҺ°еңЁеҸҜд»ҘдәҶпјҢиҝҷж ·еҮҸе°‘дәҶдёҚеҝ…иҰҒзҡ„IOж“ҚдҪңпјҢдҪҶжҳҜеҸӘиғҪз”ЁеңЁдәҢзә§зҙўеј•дёҠгҖӮ

    • Using where

      дҪҝз”ЁдәҶWHEREд»ҺеҸҘжқҘйҷҗеҲ¶е“ӘдәӣиЎҢе°ҶдёҺдёӢдёҖеј иЎЁеҢ№й…ҚжҲ–иҖ…жҳҜиҝ”еӣһз»ҷз”ЁжҲ·гҖӮжіЁж„ҸпјҡExtraеҲ—еҮәзҺ°Using whereиЎЁзӨәMySQLжңҚеҠЎеҷЁе°ҶеӯҳеӮЁеј•ж“Һиҝ”еӣһжңҚеҠЎеұӮд»ҘеҗҺеҶҚеә”з”ЁWHEREжқЎд»¶иҝҮж»Ө

    • Using join buffer

      дҪҝз”ЁдәҶиҝһжҺҘзј“еӯҳпјҡBlock Nested LoopпјҢиҝһжҺҘз®—жі•жҳҜеқ—еөҢеҘ—еҫӘзҺҜиҝһжҺҘ;Batched Key AccessпјҢиҝһжҺҘз®—жі•жҳҜжү№йҮҸзҙўеј•иҝһжҺҘ

    • impossible where

      whereеӯҗеҸҘзҡ„еҖјжҖ»жҳҜfalseпјҢдёҚиғҪз”ЁжқҘиҺ·еҸ–д»»дҪ•е…ғз»„

    • select tables optimized away

      еңЁжІЎжңүGROUP BYеӯҗеҸҘзҡ„жғ…еҶөдёӢпјҢеҹәдәҺзҙўеј•дјҳеҢ–MIN/MAXж“ҚдҪңпјҢжҲ–иҖ…еҜ№дәҺMyISAMеӯҳеӮЁеј•ж“ҺдјҳеҢ–COUNT(*)ж“ҚдҪңпјҢдёҚеҝ…зӯүеҲ°жү§иЎҢйҳ¶ж®өеҶҚиҝӣиЎҢи®Ўз®—пјҢжҹҘиҜўжү§иЎҢи®ЎеҲ’з”ҹжҲҗзҡ„йҳ¶ж®өеҚіе®ҢжҲҗдјҳеҢ–гҖӮ

    • distinct

      дјҳеҢ–distinctж“ҚдҪңпјҢеңЁжүҫеҲ°з¬¬дёҖеҢ№й…Қзҡ„е…ғз»„еҗҺеҚіеҒңжӯўжүҫеҗҢж ·еҖјзҡ„еҠЁдҪң
